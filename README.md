# Individual-Work-

## Описание
Приложение для планирования встреч с уведомлениями. Пользователи: обычный пользователь и администратор.  

## Цели
- Бронирование встреч  
- Уведомления о встречах  
- Совместимость с внешними календарями  

## Технологии
- Figma – прототип интерфейса  
- Git/GitHub – контроль версий  

## Модули
- Управление пользователями  
- Бронирование встреч  
- Уведомления  
- Интеграция с календарем

## Документация
- SRS: функциональные и нефункциональные требования  
- UML: Use Case и Activity  
- Прототип Figma
